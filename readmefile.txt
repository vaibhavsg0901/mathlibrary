I have written following functions and their discription is as follows
	fabs() function returns absolute value of the number
	exp() function returns the exponential value of number
	sine() function returns sine of the angle in radians
	cosine() function returns cosine of the angle in radians
	tangent() function returns sine of the angle in radians
	sinh() function returns hyperbolic sine of x
	cosh() function returns hyperbolic cosine of x
	tanh() function returns hyperbolic tangent of x
	asin() function returns the arc sine of x
	acos() function returns the arc cosine of x
	atan() function returns the arc tangent of x
	sqrt() function returns the squareroot of given number
	pow() function returns the value of x raised to the power of y
	fact() function returns the factorial of the given number
	log() function returns the logarithmic value  of x to the base e
	log10() fuction returns the natural logarithmic value of x
	floor() function returns largest integral value less than x. 
	myceil() function returns smallest integral value that exceeds x.
